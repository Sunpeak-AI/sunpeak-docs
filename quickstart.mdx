## New projects

<Steps>
  <Step title="Create a project">
    **Requirements:** Node.js 20+, pnpm 10+

    ```bash
    pnpm dlx sunpeak new
    ```

    You will be prompted to name your project. We'll choose `my-app` for now.
  </Step>

  <Step title="Install dependencies">
    Navigate to your project directory and install dependenceies:

    ```bash
    cd my-app
    pnpm install
    ```
  </Step>
  <Step title="Run the development server">
    ```bash
    pnpm dev
    ```

    Open `http://localhost:6767` to see the ChatGPT simulator running your app.

    <Tip>The simulator updates automatically as you edit your components!</Tip>
  </Step>

  <Step title="Build your first component">
    Edit `src/App.tsx` to create your first ChatGPT widget:

    ```tsx
    import 'sunpeak/style.css';
    import { OpenAICard } from 'sunpeak';

    export default function App() {
      return (
        <OpenAICard
          header="My First Widget"
          metadata="Built with sunpeak"
          button1={{
            children: "Click me",
            isPrimary: true,
            onClick: () => alert('Hello from sunpeak!')
          }}
        >
          This is my first ChatGPT widget built with sunpeak!
        </OpenAICard>
      );
    }
    ```

    Save the file and watch your changes appear instantly in the simulator.
  </Step>
</Steps>
