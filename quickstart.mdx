## New projects

<Steps>
  <Step title="Install the CLI">
    **Requirements:** Node.js 20+

    Install the sunpeak CLI globally to use `sunpeak` commands:

    <Tabs>
      <Tab title="pnpm">
        ```bash
        pnpm add -g sunpeak
        ```
      </Tab>
      <Tab title="npm">
        ```bash
        npm install -g sunpeak
        ```
      </Tab>
      <Tab title="yarn">
        ```bash
        yarn global add sunpeak
        ```
      </Tab>
    </Tabs>

    <Note>
    You can also use `pnpm dlx sunpeak` instead of installing globally.
    </Note>
  </Step>

  <Step title="Create a project">

    ```bash
    sunpeak new
    ```

    You will be prompted to name your project. We'll choose `my-app` for now.
  </Step>

  <Step title="Install dependencies">
    Navigate to your project directory and install dependencies:

    <Tabs>
      <Tab title="pnpm">
        ```bash
        cd my-app
        pnpm install
        ```
      </Tab>
      <Tab title="npm">
        ```bash
        cd my-app
        npm install
        ```
      </Tab>
      <Tab title="yarn">
        ```bash
        cd my-app
        yarn install
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Develop locally">
    Run the development server to use your local simulator:

    ```bash
    sunpeak dev
    ```

    Open `http://localhost:6767` to see the ChatGPT simulator running your app.

    <Tip>The simulator updates automatically as you edit your components!</Tip>
  </Step>

  <Step title="Develop in ChatGPT">
    Run the MCP server and a tunnel to allow ChatGPT to connect to it:

    ```bash
    sunpeak mcp

    # In another terminal, run a tunnel. For example:
    ngrok http 6766
    ```

    You can then connect to the tunnel `Forwarding URL` at the `/mcp` path (e.g. `https://your-random-subdomain.ngrok-free.dev/mcp`) from ChatGPT
    **in developer mode**: `User > Settings > Apps & Connectors > Create`.
  </Step>
</Steps>
