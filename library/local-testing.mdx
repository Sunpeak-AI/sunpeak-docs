## Overview

The ChatGPT Simulator is a local development environment that replicates ChatGPT's widget runtime. It lets you build and test your widgets instantly without deploying to production.

  src="https://sunpeak.ai/images/chatgpt-simulator.png"
  alt="ChatGPT Simulator"
/>

## Why Use the Simulator?

<CardGroup cols={2}>
  <Card title="Instant Feedback" icon="rotate">

## Quick Start

<Info>
If you set up your project via the sunpeak Starter Kit, the simulator is already set up!
</Info>

To learn how to manually configure the simulator, see [Add to Existing Project](/add-to-existing-project).

## What Gets Simulated

The simulator provides a complete ChatGPT-like environment including:

- **Runtime APIs**: Full access to `window.openai` including theme, display mode, and state management
- **UI Environment**: ChatGPT's chat interface, message bubbles, and widget containers
- **User Interactions**: All standard interactions like clicks, form submissions, and navigation

<Warning>
The simulator is for development only. For production testing with real ChatGPT, use the [MCP Server](/library/platform-testing).
</Warning>
