## Overview

The `OpenAICard` component is the primary building block for ChatGPT widgets, following OpenAI's design guidelines.

## Import

```tsx
import { OpenAICard } from 'sunpeak';
```

## Props

<ResponseField name="header" type="string">
  Card title/heading
</ResponseField>

<ResponseField name="metadata" type="string">
  Subtitle or additional information displayed below header
</ResponseField>

<ResponseField name="image" type="string">
  URL of the card image
</ResponseField>

<ResponseField name="imageAlt" type="string">
  Alt text for the image (required if `image` is provided)
</ResponseField>

<ResponseField name="button1" type="ButtonProps">
  Primary button configuration
</ResponseField>

<ResponseField name="button2" type="ButtonProps">
  Secondary button configuration
</ResponseField>

<ResponseField name="children" type="ReactNode">
  Card body content
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes
</ResponseField>

<ResponseField name="style" type="CSSProperties">
  Inline styles
</ResponseField>

## ButtonProps

<ResponseField name="children" type="ReactNode" required>
  Button text or content
</ResponseField>

<ResponseField name="onClick" type="() => void">
  Click handler
</ResponseField>

<ResponseField name="isPrimary" type="boolean" default="false">
  Apply primary button styling
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Disable the button
</ResponseField>

## Design Guidelines

<AccordionGroup>
  <Accordion title="Images">
    - Use high-quality images (recommended: 1200x630px)
    - Always provide `imageAlt` for accessibility
    - Images are optional but recommended
  </Accordion>

  <Accordion title="Headers">
    - Keep headers concise (1-5 words)
    - Use sentence case, not ALL CAPS
    - Headers should describe the card content
  </Accordion>

  <Accordion title="Metadata">
    - Use for supplementary information
    - Common patterns: ratings, location, price, time
    - Use emojis sparingly for visual interest
  </Accordion>

  <Accordion title="Body Content">
    - Keep to 1-3 sentences
    - Focus on key information
    - Avoid long paragraphs
  </Accordion>

  <Accordion title="Buttons">
    - Maximum 2 buttons per card
    - Primary button for main action
    - Clear, action-oriented labels ("Buy Now" not "Click Here")
  </Accordion>
</AccordionGroup>

## Styling

### Custom Classes

```tsx
<OpenAICard
  className="shadow-lg rounded-xl"
  header="Styled Card"
>
  Custom Tailwind classes applied
</OpenAICard>
```

### Theme-Aware

Cards automatically adapt to ChatGPT's theme:

```tsx
import { useTheme } from 'sunpeak';

function ThemedCard() {
  const theme = useTheme();

  return (
    <OpenAICard
      header="Theme-Aware Card"
      style={{
        borderColor: theme === 'dark' ? '#444' : '#ddd'
      }}
    >
      Adapts to {theme} theme
    </OpenAICard>
  );
}
```

## Accessibility

Cards are built with accessibility in mind:

- Semantic HTML structure
- Proper heading hierarchy
- Keyboard navigation support
- Screen reader friendly
- WCAG 2.1 AA compliant
