## Overview

`Carousel` is a production-ready carousel component included in the sunpeak Starter Kit. It provides smooth horizontal scrolling with navigation arrows, edge gradients, and persistent state tracking. Built on `embla-carousel-react` with wheel gesture support.

## Import

```tsx
import { Carousel } from './components/carousel';
import { Card } from './components/card';
```

## Basic Usage

```tsx
<Carousel showArrows={true} gap={16}>
  <Card header="Card 1">First card</Card>
  <Card header="Card 2">Second card</Card>
  <Card header="Card 3">Third card</Card>
</Carousel>
```

## Props

<ResponseField name="children" type="React.ReactNode">
  The carousel items to display. Typically `Card` components, but can be any React nodes.
</ResponseField>

<ResponseField name="gap" type="number" default="16">
  Gap between carousel items in pixels.
</ResponseField>

<ResponseField name="showArrows" type="boolean" default="true">
  Whether to show navigation arrows (Previous/Next buttons). Arrows only appear when there are more items to scroll.
</ResponseField>

<ResponseField name="showEdgeGradients" type="boolean" default="true">
  Whether to show gradient overlays at the left/right edges to indicate more content. Gradients appear when scrolling is possible in that direction.
</ResponseField>

<ResponseField name="cardWidth" type="number | { inline?: number; fullscreen?: number }" default="220">
  Width of each carousel item in pixels. Can be:
  - A single number for consistent width.
  - An object with `inline` and/or `fullscreen` properties to set different widths based on display mode.
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes to apply to the carousel container.
</ResponseField>

## State Management

The carousel uses `useWidgetState` to persist the current scroll position across sessions:

```tsx
interface CarouselState {
  currentIndex?: number;
}
```

This means users will see the same scroll position when they return to your widget.

## Common Patterns

### Product Gallery

```tsx
<Carousel gap={16} cardWidth={240}>
  {products.map(product => (
    <Card
      key={product.id}
      image={product.image}
      imageAlt={product.name}
      header={product.name}
      metadata={`$${product.price}`}
      button1={{
        children: "Add to Cart",
        isPrimary: true,
        onClick: () => addToCart(product.id)
      }}
    >
      {product.description}
    </Card>
  ))}
</Carousel>
```

### Photo Gallery

```tsx
<Carousel gap={12} cardWidth={280} showArrows={true}>
  {photos.map(photo => (
    <Card
      key={photo.id}
      image={photo.url}
      imageAlt={photo.caption}
      header={photo.title}
      metadata={photo.location}
    />
  ))}
</Carousel>
```

### Recommendations Carousel

```tsx
<Carousel gap={20}>
  {recommendations.map(item => (
    <Card
      key={item.id}
      image={item.thumbnail}
      imageAlt={item.title}
      header={item.title}
      metadata={`⭐ ${item.rating} • ${item.category}`}
      button1={{
        children: "View Details",
        isPrimary: true,
        onClick: () => navigate(item.id)
      }}
    >
      {item.summary}
    </Card>
  ))}
</Carousel>
```
