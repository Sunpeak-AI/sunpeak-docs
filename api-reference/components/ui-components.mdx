## Overview

sunpeak provides additional UI components beyond `OpenAICard` for building ChatGPT widgets.

## OpenAIButton

Standalone button component following ChatGPT design patterns.

### Import

```tsx
import { OpenAIButton } from 'sunpeak';
```

### Props

<ResponseField name="children" type="ReactNode" required>
  Button text or content
</ResponseField>

<ResponseField name="onClick" type="() => void">
  Click handler function
</ResponseField>

<ResponseField name="isPrimary" type="boolean" default="false">
  Apply primary button styling
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Disable the button
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes
</ResponseField>

### Usage

```tsx
<OpenAIButton
  isPrimary={true}
  onClick={() => console.log('clicked')}
>
  Click Me
</OpenAIButton>
```

## OpenAIList

Display lists of items with consistent styling.

### Import

```tsx
import { OpenAIList } from 'sunpeak';
```

### Props

<ResponseField name="items" type="T[]" required>
  Array of items to display
</ResponseField>

<ResponseField name="renderItem" type="(item: T) => ReactNode" required>
  Function to render each item
</ResponseField>

<ResponseField name="keyExtractor" type="(item: T) => string">
  Function to extract unique key for each item
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS classes
</ResponseField>

### Usage

```tsx
const items = [
  { id: 1, name: 'Item 1', description: 'Description 1' },
  { id: 2, name: 'Item 2', description: 'Description 2' },
];

<OpenAIList
  items={items}
  renderItem={(item) => (
    <div>
      <h3>{item.name}</h3>
      <p>{item.description}</p>
    </div>
  )}
  keyExtractor={(item) => String(item.id)}
/>
```

## Additional Components

More components are available from the underlying `@openai/apps-sdk-ui` library:

```tsx
import {
  Badge,
  Input,
  Select,
  Checkbox,
  Radio,
  Switch,
  // ... and more
} from '@openai/apps-sdk-ui';
```

See the [apps-sdk-ui documentation](https://openai.github.io/apps-sdk-ui/) for complete reference.
